# 🎓 AI-ML Capstone Project

Student Performance Prediction using Machine Learning and Explainable Generative AI.

---

# 📌 Project Overview

This capstone project demonstrates an end-to-end Machine Learning workflow for predicting student performance using structured educational data.

The project covers the complete ML lifecycle including:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Regression & Classification Models
- Ensemble Learning
- Hyperparameter Tuning
- Model Serialization
- Explainable AI using Google Gemini
- Production-ready ML Pipeline

The project is divided into four independent parts, each focusing on a different stage of the Machine Learning pipeline.

# ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/hasanhafeel24/AI-ML-Capstone.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Navigate to any project part:

```bash
cd PART-4
```

Run the application:

```bash
python app.py
```
