# PART 2 - Model Training

## Objective
Train a Machine Learning model to predict student exam scores.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Joblib

## Workflow

### 1. Data Loading
- Loaded cleaned student dataset.

### 2. Data Preprocessing
- Converted categorical columns using One-Hot Encoding.

### 3. Train-Test Split
- 80% Training Data
- 20% Testing Data

### 4. Model Training
- Linear Regression

### 5. Model Evaluation

Results:

- MAE : 0.45
- MSE : 3.26
- RMSE : 1.80
- R² Score : 0.77

### 6. Outputs

Generated files:

- models/student_performance_model.pkl
- results/predictions.csv
